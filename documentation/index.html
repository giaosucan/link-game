Importing code into Eclipse

To import the code into your eclipse do the following steps
  1. Click on File
  2. Select New Project
  3. Select Android Project
In the dialog, choose the option "Existing Project" and put the path to the  project root path.

Eg - you have put code in C drive then put the path as Local Disk C
In the following list you will be shown the list of projects that you can select Then select Link.

Complete Offers to get Hints

Completing offers to get in app items are being used extensively in most of the apps from a quite a while back. We have also implemented tapjoys' offer wall into the source code. Its very simple. Whenever a user wants to get more hints he/she could complete a free offer from the offerwall and get hints in written. Offers usuaaly include downloading another app.

What you have to do?

  1. Register yourself at tapjoy ( If you are not already)  < dashboard.tapjoy.com >.
  2. From the dashboard at tapjoy create a new app and get a new app id.
  3. Now create a new virtual currency from the virtual currency option present under the publish.
  4. Now from the General section go to details and get your app id, app key.
  5. Hint key can also be obtained but it will be similar to your app id so no need for that.

Note - To access tapjoy offerwall you just have to mail them with your id also saying you want to use thier offerewall for providing free hints so that they should activate your id.

Replacing the Tapjoy IDs in the source code - 

  1. Go to res-->values-->string.xml.
  2. There you will see a section starting with  <!-- values for tapjoy free hints --> 
  3. Just replace the respective ids with your ids.
  4. Voilla your tapjoy is ready to earn money.

I DONT NEED OFFERWALL - 

In case you dont need offerwall then do the following steps and get rid of it.

  1. Go to src-->com-->creativedroids-->link-->utlis-->DataHandler.java.
  2. There you will see a variable named tapjoy_ads, just set its value to false.

Vungle Video Ads

About Video Ads

Now a days many succesful and top developers have started using video ads in their application because of their very high ecpm in comparision to banner and interstitial ads. Video ads offers you to earn more with your app by providing users a rich media type ads.
We too have implemented the video ads in the game as a means to earn to hints. Its like whenever a user watches a video he/she gets 5 extra hints. For this purpose we have incorporated vungle sdk into the app.

What you need to do?

  1. Register yourself at vungle as a developer ( if you are already not registered) < www.vungle.com >.
  2. Create your application.
  3. Get your vungle app id.

After getting your vungle app id just replace your vungle app id with our id given in the source code.

Replacing Vungle App id -

  1. Go to res-->values-->strings.xml.
  2. There you will see a line like this   <string name="appId">creativedroidsapp001</string>
  3. Replace our id with your own

Note - In case of vungle, unlike the rest of other ad serving companies, you have to create your own unique id.

Modifying Number of hints given on watching a video ad - 

  1. Go to res-->values-->integer.xml.
  2. There you will see a line like this   <integer name="videoHints">5</integer>
  3. Change this 5 to anything you want.

I DONT NEED VIDEO ADS - 

In case you dont need video ads then do the following and get rid of it.

  1. Go to src-->com-->creativedroids-->link-->utlis-->DataHandler.java.
  2. There you will see a variable named video_ads, just set its value to false.

Removing Errors (If any)

If you recieving any errors most likely R cannot be resolved to a variable or android cannot be resolved to any type then do the folllowing
  1. Right Click on your folder in package explorer in eclipse
  2. Slect properties
  3. Then select Android
  4. Then in the android build target folder select the latest sdk you have.
  5. Select OK.
If you face any other type of errors related to google play libraries or admob then do the following step.
 
  1. Right Click on your folder in package explorer in eclipse
  2. Slect properties
  3. Then select Android
  4. On the bottom part of the dialog box make sure there is a green tick in front of the google play library instead of red cross.
  5. Select OK.
If you are facing red cross then remove the whole project and reimport it again. Eclipse sometimes dont take library files but redoing it will solve your problem. If problem still persists do mail us from our profile page

http://codecanyon.net/user/CreativeDroids
